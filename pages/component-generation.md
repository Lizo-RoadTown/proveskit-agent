# Component Generation (Planned)

The Builder Agent for F Prime component generation is planned but not yet active. Current development focus is on the curator workflow, truth layer, and MCP server.

---

## Planned Capabilities

- Generate FPP component definitions from validated patterns
- Produce C++ scaffolds aligned with F Prime conventions
- Attach citations and evidence from the knowledge graph
